
AI Email Classifier is a project for IIT Bombay competition.

This project has re-trainable models where we can add classes and examples to re-train the models to perform multi-class
classification on the text data.

Modules:

1. ai_classifiers

This package has all the classifiers that we will use for classification, NLP processing, etc.

2. ai_metrics
Evaluation metrics.


3. frontend
Logic for front end using streamlit library.


4. preprocessing
converting zip dataset to csv, etc.
